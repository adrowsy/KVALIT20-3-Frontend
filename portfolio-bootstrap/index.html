School assignments

Samples shown below are, if not otherwise stated, results from school assignments.

Java application: Notebook

A to do-list inspired by the Bullet Journal method. Allows the user to import notes from file and save data to an object array. User can read the notebook and change note type to completed.

  • 3 classes and 14 methods
  • Private instance variables, using instance methods
  • Error handling and data validation

Java application: Dice

Presents the result from 1 000 dice rolls using three methods:

  • createFile() returns a file with 1 000 random values 1 to 6
  • creatArray() returns an int array with file values
  • analyseArray() count and sort int array, returning number of 1s, 2s, 3s, etc.

Accessibillity testing

Test object: Swedish municipal web diary

Accessibillity testing based on the Act on Accessibility to Digital Public Services with a special focus on documents (.pdf) distributed via the website. The law has been in force since 1 January 2019 for all public websites in the EU. The scope has been adapted to the time limits when different parts of the Web Accessibility Directive must be met.

  • Individual project, 20 hours
  • Focusing on capability, reliability och usability limited to WCAG A‑levels 1.1.1, 1.3.1, 1.3.2 and 1.3.4
  • Deliverables: STP, STD, STC, STCR, STR, SDR, STS
